# SMB Risk Dashboard (Demo)

A lightweight Streamlit app that shows:

* Customer‑sentiment (VADER) on e‑mail & chat messages  
* Synthetic credit‑card transactions (amount, category, over‑limit flag)  
* Flags for NSFW, financial‑misuse, compliance, personal‑use  

The UI follows the **Tiffany‑blue** style you requested and displays your company logo.

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
